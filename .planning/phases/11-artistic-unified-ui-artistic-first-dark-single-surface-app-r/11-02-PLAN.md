<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| microphone → app | getUserMedia grants live mic access |
| user file → API | arbitrary local file uploaded as a Blob |
| API responses → DOM | notes/messages rendered into overlays and whispers |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-03 | Information Disclosure | MediaRecorder adapter | mitigate | stop all MediaStream tracks on stop/abort/unmount; recording state always visible in the sheet |
| T-11-04 | Tampering | file drop/browse input | mitigate | accept="audio/*" filter; file travels as multipart Blob through the existing seam; bytes never interpolated into DOM or URLs |
| T-11-05 | Spoofing/XSS | error messages + notes rendered in whispers/banners | mitigate | text nodes only; whisper region renders strings, never markup |
| T-11-SC | Tampering | npm installs | mitigate | no installs in this plan; lockfile unchanged (covered by 11-01 Task 1 gate) |
</threat_model>
