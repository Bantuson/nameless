# Quick Task 260704-0v6: Add GitHub Actions CI Workflow Summary

**One-liner:** Four-job GitHub Actions CI (rust-lean, rust-postgres with migrated pgvector:pg16 for sqlx compile-time verification, uv/pytest matrix over workers + knowledge-pipeline, npm ci + vitest + tsc/vite build) — the verification vehicle for everything STATE.md marks env-gated.

## What Was Built

`.github/workflows/ci.yml` (141 lines) — the repo's first workflow, triggered on push to `main`, PRs to `main`, and manual dispatch. Top level: `permissions: contents: read` (no secrets anywhere; PR-safe), concurrency group `ci-${{ github.ref }}` with cancel-in-progress. All jobs on `ubuntu-latest`, pure siblings with zero inter-job dependencies:

1. **rust-lean** (30 min timeout) — first-ever `cargo build --workspace` + `cargo test --workspace` on the lean default feature set. No lockfile flags (no `Cargo.lock` exists; first build generates it).
2. **rust-postgres** (45 min) — `pgvector/pgvector:pg16` service (nameless/nameless/nameless, port 5432, pg_isready health checks); job-level `DATABASE_URL` so it exists both when migrations run and at cargo compile time (sqlx `query!`/`query_as!` macros connect during compilation — no `.sqlx` offline cache in this repo); psql loop applies `migrations/*.sql` (0001→0004, lexicographic) with `ON_ERROR_STOP=1` strictly before any cargo step; build + test use virtual-workspace pkg/feature syntax `nameless-adapters/postgres,nameless-cli/postgres,nameless-api/postgres`.
3. **python** (15 min) — `fail-fast: false` matrix over `[workers, knowledge-pipeline]`; `astral-sh/setup-uv@v6` with per-project pyproject cache glob; `uv sync --python 3.12 --extra dev` (light tier only — no ML/network extras); `uv run pytest`.
4. **web** (15 min) — `actions/setup-node@v4` (node 22, npm cache keyed on `web/package-lock.json`); `npm ci` → `npm test` (`vitest run`, non-watch) → `npm run build` (`tsc --noEmit && vite build`, covering typecheck + production build in one step).

Actions pinned to major tags only: `actions/checkout@v4`, `dtolnay/rust-toolchain@stable`, `Swatinem/rust-cache@v2`, `astral-sh/setup-uv@v6`, `actions/setup-node@v4` — no `@main`/`@master` refs, no other actions.
